Oppsummering av video:

Om print () heltall, desimaltall og string.

I videoen snakker vi om print () funksjonen.

Vi kan se på print som en kalkuleringskommando som vi bruker for at Python skal gi oss svarene på de utregningene vi gjør.

Når vi bruker print legger vi alt som vi ønsker å skrive ut inni en parentes.

I kodefeltet skriver vi hva vi ønsker å regne ut (input)

Når vi trykker på 'Run' ("play"-knappen øverst) regner Python ut det vi ønsker å regne ut og sender svaret til konsollen

Eksempel

input (skrevet i kodefeltet):

print (20*20)

Output (dukker opp i konsollen):

400

Python kan printe ut tekst, heltall og desimaltall.

I Python har heltall, desimaltall og setninger andre navn enn vi er vant til:

Heltall = int (forkortelse for integer som betyr heltall)

Desimaltall = str (forkortelse for float numbers)

Setninger = str (forkortelse for string-variabel, altså en snor av symboler)

Heltall skriver vi som på en vanlig kalkulator, mens for deismaltall bruker vi punktum i stedet for komma

Dersom vi ønsker å bruke tekst må vi ha anførselstegn foran og bak det vi skal skrive:

Input:

print ('hei')

Output:

hei

Når vi skal forstå forskjellen kan vi nøye oss med å si at alt som er mellom anførselstegn er en del av en string.

Altså er '5' en string, mens 5 er et heltall.

Om Python og trinket.io

Det finnes mange ulike programmeringsspråk. Språket man bruker er avhengig av hva du vil lage

Python er et av de vanligste språkene for utviklng av kunstig intelligens og Chat GPT er primært skrevet i Python.

Python kan være med å gjøre algebra, likninger og funksjoner enklere å forstål!

Trinket er en web-basert python-editor, som vil si at du kan lage og skrive ut koder uten at du trenger å laste ned noen ting.

Trinket-vinduet er delt i to på midten (vertikalt/loddrett). Venstre siden av vinduet er der du skriver koden din.

Dette kaller kodefeltet. Du kan se at linjene nummereres etterhvert som koden blir til. Dette gir deg oversikt over koden din.

Høyre side kaller vi konsollen, og det er her alt du printer ut og alle feilmeldinger dukker opp.

En feilmelding er en tilbakemelding om at noe i koden er feil. I feilmeldingen står det besrkevet hva slags feil det er og ved hvilken linje feilen er.

Du kan justere størrelsen på de to sidene ved å dra skillelinjen til høyre eller venstre.
