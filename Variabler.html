Variabler:

Variabler brukes for:

- Å gi navn til ulike verdier og elementer i koden.

- Å ha muligheten til å justere verdier som går igjen flere steder i koden

Når du kjører programmet ditt vil en variabel som defineres flere ganger endres underveis.

Eksempel:

Input:

mitt_tall = 12

print ('første gang:',mitt_tall)

mitt_tall = 9

print ('andre gang:',mitt_tall)

Output:

første gang: 12

andre gang: 9

Regning med variabler:

I python kan vi gi variabler tallverdier.

Når flere variabler består av heltall (int) eller desimaltall (float) kan vi regne med variablene. Du kan sette opp regnestykke med variablene, og endre verdiene på dem for å gjøre samme regneoperasjon med ulike verdier.

Eksempel:

Input:

a = 3

b = 7

print ('a + b = ',a + b)

b = 9

print ('a + b = ', a + b)

Output:

a + b = 10

a + b = 12

Legg merke til forskjellen mellom 'a + b =' og a + b

Når anførselstengene står mellom tolker Python dette som en string. Dersom vi ikke hadde hatt med 'a + b =', ville vi kun fått oppgitt svaret i konsollen.

Input:

a = 3

b = 5

print (a + b)

Output:

8
